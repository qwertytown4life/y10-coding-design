Welcome to Coding Ninja!

Coding Ninja is an interactive website that allows you to learn the basics of coding in python3! In this website you will cover the topics of basic arithmetic, if statements, loops, and functions. These skills are widely applicable to programming across all languages. To learn these skills, click on the 'Lessons' button. To find problems to test your skills, click on the 'Problems' button. To find more practice problems, click on the 'Extra Practice' button.
