Functions

In programming, functions are self contained pieces of code that 'do things' when certain parameters are given to them. In python a function is defined as:

def function(x):

The above code is a defining a function named 'function' that passes a something for the parameter 'x'. Anything after that must be indented. The end of a function must end with a return statement.

An example of a function that takes in a number 'x' and doubles it would look like:

def function(x): return x*2


If-Statements

Programming is filled with decisions. If we want to make decisions about certain things based on information, we must you an if-statement. In python you can write them as:

if (condition): do something

You can also use else if (elif) or else (else) statments.For example, lets define 6 to be a special number. If we want to write an if statement that checks if a number is equal to 10 or not.

if == 10: return 'number is equal to 10' else: return 'number is not equal to 10'

*Note that to check if two things are equal you must use '==' and not '=' as the latter is an assignment operator.


Simple Arithmetic

Math is frequently used in programming. Below are the basic operators:

Math Operation Python Symbol
addition +
subtraction -
multiplication *
division /
exponentiation **
modulus (remainder in division) %

Note that python follows order of operations. For example:

x = 4*3 + 2 - 9/3**2 x = (12) + 2 - (9/9) x = 14 - (1) x = 13


Variables and Arrays

In python you can store information in variables and arrays. Below is declaring a variable named 'x' and setting it equal to 5.

x = 5

Notice that a single equals sign is used. Below is setting an array of values equal to [1, 4, 6]. An array is a contained 'list' of values held together. You can access them individually by typing arrayName[i], where arrayName is the name of the array and i is the index of the element you want to access. Note that arrays are 0-indexed. In other words, they start at 0 and count of until the lenght of the array minus 1.

x = [1,4,6]


Loops

In order to iterate through things, you must you loops. There are two loops in python. While loops and for loops. Below is an example of a while loop:

while (condition is true): (do something)

For loops are a little harder and look like:

for ([starting point], [ending point], [increment/decriment]): do something
